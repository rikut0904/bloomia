# AIDEMIA README

## システム概要

AIDEMIAは、GIGAスクール構想以降のICT教育を支援する新たな学習管理システム(LMS)です。学習管理機能に加え、教師と生徒、生徒同士の活発なコミュニケーションを支援する機能や、学習意欲を高めるためのゲーム性を組み込んだ次世代LMSです。

これまでの学校現場で複数のツールが併用されていた状況を改善し、本アプリ一つで教材共有・課題提出・コミュニケーション・成績管理まで完結できることを目指しており、「これ一つあれば十分」と思えるような統合型の学習プラットフォームを提供します。

## 作成意図

AIDEMIAは、単なる「管理ツール」ではなく、「生徒が継続して学べる」「教師が現場で使いたくなる」LMSを目指して開発しています。

### 本プロジェクトの3つの柱：
1. *コミュニケーションの活性化*
　チャットや通知機能により、質問・回答・共有が即座に行える環境を提供します。

2. *モチベーションの継続支援*
　課題達成に応じたポイント付与やランキング表示により、楽しみながら取り組める仕組みを用意します。

3. *オールインワン設計による効率化*
　教材閲覧、課題管理、出欠確認、成績管理を一つのアプリに集約します。

## 使用技術

* フロントエンド: Next.js, Vercel
* バックエンド: Rust, Railway, Redis
* データベース: PostgreSQL (Railway)
* ストレージ: Firebase Cloud Storage
* 認証: JWT, OAuth
* 開発環境: Ubuntu (WSL2), VSCode, Docker

## 技術選定理由（URL付き）

### [Next.js](https://nextjs.org/)
複雑なUIの構築を効率化。

### [Vercel](https://vercel.com/)
Next.jsとの親和性が高いため。

### [Rust](https://www.rust-lang.org/)
パフォーマンスとセキュリティを重視し、高信頼なサーバ実装が可能であるため。

### [Railway](https://railway.app/)
デプロイやデータベース運用の簡便さを活かし、開発スピードを加速。

### [Redis](https://redis.io/)
キャッシュ管理による負荷分散と応答速度高速化のため。

### [Firebase Cloud Storage](https://firebase.google.com/products/storage)
メディア・ドキュメント共有のためのストレージ運用をおこなうため。

### [JWT](https://jwt.io/), [OAuth](https://oauth.net/)
ログイン認証に利用。

### [Docker](https://www.docker.com/)
開発環境間の環境差異を減らし、効率的な開発・デプロイを実現。

## 主な機能

### 生徒用

* **教材閲覧**: PDF/プレゼン資料の閲覧
* **課題提出**: 記述式/ファイル提出/多指選式
* **課題通知**: 期限前のリマインド
* **進捗管理**: 終了/未終了課題の総計
* **成績管理**: チャート表示での分析
* **時間割閲覧**: 次回の講義内容を描示
* **出答管理**: 出席/避け/欠席の分類表示

### 教師用

* **教材共有**: バージョン管理機能つき
* **課題の出題・回収**: リマインドつき提出状況を表示
* **進捗の一括管理**
* **成績登録/管理**
* **出答登録/確認**

### 共通機能

* **SNS機能**: メッセージ/コメント/いいね/チャットルーム
* **ファイル共有**: 課題に関連する資料の插入
* **ログ管理**: 利用記録、エラーをTXTでローカルサーバーに保存

### 発展機能

* **ポイント管理**: 課題終了ごとのポイント付与 + ランキング
* **教材にメモ/マーカー機能**
* **共有ノート**
* **音声読上げ機能**
* **テンプレート/カスタマイズ配信**
* **個別通知機能**
* **リアクション/同期編集/レポート自動生成**

## データベース構成

付箋のデータベース構成図を参照。実装はリレーショナルDB構造で管理され、下記のようなモジュールで構成されています:

* 個人情報DB: 生徒/教師情報、成績、出席
* 課題DB: 課題情報、提出状況
* SNS DB: ポスト、コメント、メッセージ、チャットルーム
* 連続ストレージ: Firebase を用い、チャットログ/メディアが存儲
